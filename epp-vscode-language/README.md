# E++ VS Code Language Support

This local extension adds:

- `.epp` language recognition
- syntax highlighting for E++ keywords, numbers, strings, comments, and built-ins
- line comment support with `#`

## Install in VS Code

1. Open VS Code.
2. Press `Ctrl+Shift+P`.
3. Run `Developer: Install Extension from Location...`.
4. Pick this folder: `epp-vscode-language`.
5. Reload VS Code.

After install, open any `.epp` file and the language mode will be `E++`.
